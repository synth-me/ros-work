<h1>Cada um desses arquivos tem uma função, as quais são </h1>

  * **my_node** → Nó simples 
  * **SpiderPublisher** → O publisher ja com as caracteristicas necessarias desejadas 
  * **SpiderSubs** → Le o output do SpiderPublisher
  * **SpiderTurtle** → Controla a tartaruga da maneira desejada e publica as informações para o monitor
  * **SpiderMonitor** → Monitora a posição em X,Y,Z da tartaruga . Pegando as informações do arquivo SpiderTurtle
